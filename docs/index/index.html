Finscrapers scrapes financial data from the web in seconds.

Any data

Connect with any data source, with or without API. Thanks to the Puppeteer integration, you can source financial data from websites directly. Join our community to build new connectors.

Flexible processing

Finscrapers delivers the financial data through a REST (coming: GraphQL) endpoint. From here you can take it to Excel, a database (roadmap), or integrate it in your own custom application, or trading bot.

Free, and open-source

Built for the community, Finscrapers strives to become the open standard to mine financial data, and build custom applications for traders, analysts, and investors.

Integrated modeling

Gain insights from data with the integrated financial modeling capabilities (roadmap) that enables you to easiyl calculate company valuations in an automated way.
